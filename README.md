# Catalogo de Llaveros Personalizados

Breve descripcion del repositorio Desarrollo-web:
Este repositorio reune practicas basicas de desarrollo web (HTML y CSS) orientadas a proyectos reales y publicables en GitHub Pages.

Este proyecto contiene un catalogo sencillo de llaveros personalizados para Bogota, Colombia, pensado para estudiantes con conocimientos basicos.

## Que incluye este proyecto

- Landing page responsive con enfoque comercial.
- Catalogo de productos con precios base en COP.
- Flujo de pedido por WhatsApp.
- Ilustraciones vectoriales (SVG) de cada tipo de llavero.

## Estructura

- `index.html`: contenido principal del catalogo.
- `styles.css`: estilos visuales y responsive.
- `llavero-nombre.svg`
- `llavero-foto.svg`
- `llavero-empresa.svg`
- `llavero-resina.svg`

## Productos de ejemplo

- Llavero Nombre
- Llavero Foto
- Llavero Empresarial
- Llavero en Resina

## Publicar en GitHub Pages

1. Crear un repositorio en GitHub, por ejemplo: `desarrollo-web-llaveros`.
2. Subir todos los archivos en la rama `main`.
3. Ir a `Settings > Pages`.
4. En `Build and deployment` seleccionar:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main` y carpeta `/(root)`
5. Guardar y esperar 1-2 minutos.
6. Abrir la URL publica generada por GitHub Pages.

## Edicion rapida para estudiantes

1. Cambiar marca y textos en `index.html`.
2. Ajustar colores en variables `:root` dentro de `styles.css`.
3. Reemplazar numero de WhatsApp en el boton de contacto.
4. Editar precios segun tipo de cliente (unidad o mayorista).
5. Sustituir los SVG en la raiz del proyecto por disenos propios.

## Ruta practica de trabajo

1. Elegir un nicho: regalos, eventos, emprendimientos o colegios.
2. Preparar un catalogo simple con 4 productos iniciales.
3. Definir precio por unidad y por cantidad.
4. Publicar en GitHub Pages y compartir el enlace como portafolio.
5. Tomar pedidos por WhatsApp con mensaje corto y claro.
